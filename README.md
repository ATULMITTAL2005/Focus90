# Focus90

A minimal React + Tailwind app scaffold for a 90-day focus & phone detox system.

Quick start:

1. npm install
2. npm run dev

Features: Dashboard, 90-day roadmap, Phone Detox Tracker, Task Manager, Focus Mode, Progress Analytics, Motivational system.

This scaffold uses Vite + React + Tailwind (no backend).
